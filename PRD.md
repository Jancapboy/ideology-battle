# PRD — 概念大逃杀 (Ideology Battle Royale)

## 1. 产品定位

**一句话**：把抽象概念/意识形态/社会思潮变成自走棋单位，让它们在一个封闭的战场上自动厮杀，最后看谁的理念能统治全场。

**目标用户**：
- 喜欢玩自走棋/增量游戏的人
- 对社会科学、哲学、网络文化感兴趣的人
- 想看点荒诞又有梗的内容创作者

**核心价值**：
- **娱乐性**：看"资本主义"被"躺平"反噬，"内卷"和"人工智能"联手屠杀全场
- **话题性**：每场对战都是天然的内容素材（截图/录屏/直播）
- **策略性**：羁绊组合、站位、装备搭配有深度

---

## 2. 核心玩法

### 2.1 单位设计

每个"概念"是一个战斗单位，拥有：

| 属性 | 说明 |
|------|------|
| 名称 | 如：资本主义、社会主义、躺平、内卷、AI、DAO、996、远程办公 |
| 生命值 (HP) | 概念的"社会接受度" |
| 攻击力 (ATK) | 概念的"传播力/冲击力" |
| 攻击速度 (SPD) | 概念的" viral 程度" |
| 阵营/标签 | 如：经济系、政治系、技术系、反乌托邦系 |
| 技能 | 每个概念有一个主动或被动技能 |

### 2.2 技能示例

| 概念 | 技能名 | 效果 |
|------|--------|------|
| 资本主义 | 市场无形之手 | 每 5 秒随机给一个友军加 ATK，但有 20% 概率让最穷的友军直接破产（即死） |
| 躺平 | 低欲望护盾 | 受到的第一次致命伤害无效，但 ATK 永久 -30% |
| 内卷 | 零和博弈 | 攻击时偷取目标的 SPD 给自己 |
| AI | 指数增长 | 每击杀一个敌人，ATK 和 HP 上限 +15% |
| 996 | 透支 | 开场 ATK x2，但每 10 秒损失 10% HP |
| 远程办公 | 分布式抗性 | 受到的 AOE 伤害 -50% |

### 2.3 战斗规则

- **8x8 网格战场**
- **回合制自动战斗**（类似自走棋）
- 每回合：
  1. 双方各上场 5-8 个单位
  2. 自动寻敌、普攻、释放技能
  3. 直到一方全灭
- **羁绊系统**：同标签达到 2/4/6 个时触发群体 buff
  - 经济系：金币掉落 +20% / +50% / +100%
  - 反乌托邦系：全员开场获得护盾
  - 技术系：每回合随机升级一个单位的星级

### 2.4 成长与经济

- **金币**：胜利、击杀、连胜/连败奖励
- **升级商店**：用金币刷新/购买新概念、合成三星单位
- **装备/思潮**：给单位装备"短视频算法"（+SPD）、"学术期刊"（+ATK，-SPD）、"模因病毒"（死亡时爆炸传染）等

## 3. 产品形态

**第一阶段（MVP）**：
- 纯 Python 命令行/终端可视化
- ASCII/Unicode 战场 + 文字战斗日志
- 本地可运行，无需前端

**第二阶段（可选）**：
- Web 界面（React + Canvas/WebGL）
- 可分享对战回放链接
- 用户自定义概念上传

---

## 4. 内容扩展

### 4.1 概念图鉴（首批 30 个）

分 6 大阵营，每阵营 5 个单位：

| 阵营 | 示例单位 |
|------|---------|
| 经济系 | 资本主义、社会主义、凯恩斯主义、奥派、MMT |
| 政治系 | 民主、专制、无政府、威权、民粹 |
| 技术系 | AI、区块链、元宇宙、赛博朋克、生物技术 |
| 社会系 | 内卷、躺平、996、远程办公、 FIRE |
| 文化系 | 消费主义、复古怀旧、亚文化、主流审美、取消文化 |
| 反乌托邦系 | 老大哥、监控资本主义、算法推荐、信息茧房、气候危机 |

### 4.2 特殊模式

- **历史重演模式**：固定阵容（如 1929 大萧条 vs 罗斯福新政）
- **网络热梗模式**：只用当年的流行概念（如 2024：Sora、脑机接口、县城文学）
- **自定义模式**：用户输入任意词，AI 生成它的属性/技能/标签

---

## 5. 技术关键点

- 自走棋 AI 寻敌逻辑（最近敌人 / 最低 HP / 克制关系）
- 技能效果系统（buff/debuff/延迟触发/条件触发）
- 战斗回放序列化（用于分享和复盘）
- 平衡性：数值模拟 + 蒙特卡洛胜率测试

---

## 6. 成功指标

- MVP 能在终端跑一场 8 人混战
- 30 个概念各有独特技能且无明显废物
- 存在至少 3 套 viable 的吃鸡阵容
- GitHub 仓库 CI 通过 + 有集成测试覆盖对战流程
